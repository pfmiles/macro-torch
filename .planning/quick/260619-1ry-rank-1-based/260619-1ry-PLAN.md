# Quick Task 260619-1ry: 技能释放默认最高等级 + 可选rank参数

**Date:** 2026-06-18
**Status:** Ready for execution

## Task Description

修改法术释放系统：
1. 默认释放**最高等级**技能（而非当前的最低等级）
2. 支持可选的 `rank` 数字参数指定等级（1-based，法术书中同一法术的出现顺序）
3. 不传 rank → 最高等级；传入超过最大等级的 rank → 最高等级

## Root Cause

`getSpellIdByName` (`biz_util.lua:21`) 从 i=1 向上遍历法术书，返回**第一个**匹配（最低等级）。目标型技能通过 `CastSpell(id, 'spell')` 释放，实际释放了最低等级。

## Key Design Decisions

1. **rank 参数放在 mode 参数之后**：保持 API 兼容性，`wrath(nil, 3)` 或 `wrath('ready', 3)` 都可工作
2. **self-cast 路径也支持 rank**：使用 `CastSpellByName("Wrath(Rank 3)", true)` 格式
3. **不修改 `getSpellIdByName` 原有行为**：它可能被其他地方用于检查法术是否存在（如 `isSpellExist`），改为新建函数
4. **`isSpellReady` 检查需要适配 rank**：指定 rank 时，`SpellReady("Wrath")` 可能不够精确，需要检查带 rank 后缀的法术名
