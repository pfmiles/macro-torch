创建 core/selftest.lua，包含 Apache 2.0 许可证头，然后实现以下三个模块：

**模块 1: SelfTest 表初始化 (per D-01, D-03, D-05)**
- 初始化 `macroTorch.SelfTest = { tests = {} }`
- session flag `macroTorch._selfTestRan = nil`

**模块 2: SelfTest:register(name, fn, isOptional) (per D-03, D-05)**
- 将 `{name=name, fn=fn, isOptional=isOptional or false}` 插入 `self.tests` 表
- isOptional 默认 false

**模块 3: SelfTest:run() (per D-01, D-05)**
- 检查 `macroTorch._selfTestRan`，若为 true 直接 return
- 设置 `macroTorch._selfTestRan = true`
- 遍历 `self.tests`，对每个 test 执行 `pcall(test.fn)`，按以下逻辑收集结果：
  - pcall 成功 → passed++，不打印任何日志
  - pcall 失败 + isOptional → warnings++，记录 name 到 warningNames
  - pcall 失败 + not isOptional → failed++，记录 name 到 failedNames
- pcall 失败时捕获错误详情 (per RESEARCH Pitfall 5): `"[macro-torch] FAIL: " .. name .. " - " .. tostring(err)`
- 输出汇总行: `macroTorch.show(string.format("[macro-torch] Self-test: %d passed, %d failed, %d warnings", passed, failed, warnings), 'white')`
- 遍历 failedNames 输出 `"[macro-torch] FAIL: " .. name` 到聊天框，红色
- 遍历 warningNames 输出 `"[macro-torch] WARN: " .. name` 到聊天框，黄色

**模块 4: /mt SLASH 命令 (per D-02, D-10)**
- 注册 `SLASH_MT1 = "/mt"`
- 注册 `SlashCmdList["MT"] = function(msg)` 处理函数
- msg 去除首尾空格后若为空字符串或 nil → 调用 `macroTorch.SelfTest:run()`
- msg 非空 → 输出提示 `"[macro-torch] /mt: mt-script DSL is reserved for a future phase. Use /mt without arguments to run self-test."` (黄色)，预留扩展点

不要创建任何其他文件。所有实现放在 core/selftest.lua 中。SelfTest:run() 不使用递减计数器的早期模式 — 用简单的 passed/failed/warnings 累计。
