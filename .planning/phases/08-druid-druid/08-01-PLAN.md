<!-- Key types and contracts the executor needs. Extracted from codebase. -->

From core/class.lua:
- `macroTorch.classMetatable(cls, fieldMapName)` — creates metatable with __index chain: FIELD_FUNC_MAP -> cls -> parent chain
- `macroTorch.registerPlayerClass(className, classTable)` — registers class in PLAYER_CLASS_REGISTRY
- `macroTorch.PLAYER_CLASS_REGISTRY` — table for polymorphic initPlayer lookup

From entity/Player.lua:
- `obj._castSpell(localeNames, mode, range, resourceCost, onSelf)` — shared spell casting with locale support, readiness check, resource check
  - localeNames: { en = 'EnglishName', zh = '中文名' }
  - mode: nil='ready' (check isSpellReady), 'raw'=no checks, 'safe'=all checks (distance+resource)
  - range: nil = melee (skip distance check), number = range in yards
  - resourceCost: nil = skip, number = fixed cost, function = dynamic cost
  - onSelf: boolean, true = cast on self
- `obj.cast(spellName, onSelf)` — simple spell cast (legacy wrapper)
- `obj.isSpellReady(spellName)` — cooldown check via GetSpellCooldown
- `macroTorch.castIfBuffAbsent(unit, spellName, texture)` — cast spell only if buff absent on unit (keep for Warlock/Priest/Mage/Warrior patterns)
- `macroTorch.castSpellByName(name, 'spell')` — raw WoW API wrapper

From core/spell_trace_core.lua:
- `macroTorch.SpellTrace:register(name, config)` — declarative spell trace registration
  - config.immune: bool, whether to track immunities
  - config.land: bool, whether to track land events
  - config.debuffTexture: string, debuff icon texture path
  - config.spellId: number (optional)

From core/selftest.lua:
- `macroTorch.SelfTest:register(name, fn, isOptional)` — register self-test case
  - fn is wrapped in pcall; guard with UnitClass('player') check for class-specific tests
  - isOptional: true for class-specific tests (non-core)

From the Druid reference pattern (classes/druid/Druid.lua):
- Constructor uses `function obj.skill_name(mode)` closures with `obj._castSpell({en='...', zh='...'}, mode, range, cost, onSelf)`
- FIELD_FUNC_MAP is a global table `macroTorch.XXX_FIELD_FUNC_MAP = { ... }` with lazy-computed property functions
- Singleton: `macroTorch.xxx = macroTorch.Xxx:new()`
- Registration: `macroTorch.registerPlayerClass("ClassName", macroTorch.Xxx)`
- SpellTrace calls `SpellTrace:register()` with spell name, config
- SelfTest calls `SelfTest:register()` with UnitClass guard + isOptional=true
