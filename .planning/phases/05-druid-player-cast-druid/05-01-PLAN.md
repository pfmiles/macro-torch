<objective>
Add three shared spell casting helper methods to the Player base class in entity/Player.lua:
1. `_castSpell(localeNames, mode, range, resourceCost, onSelf)` — locale-aware spell casting with mode-driven readiness/distance/resource checks
2. `_isInRange(range)` — checks if current target is within casting range
3. `_hasResource(cost)` — checks if player has sufficient energy/rage/mana

Purpose: Establish the foundational infrastructure that all ~43 Druid skill methods (Plan 05-02) and ~32 call-site replacements (Plans 05-03, 05-04, 05-05) depend on. The `_castSpell` method encapsulates locale selection, readiness checking, distance checking, and resource checking in a single shared helper, eliminating duplicate logic across 40+ skill methods.

Output: `entity/Player.lua` modified — three new methods added to `Player:new()` constructor body.
</objective>
