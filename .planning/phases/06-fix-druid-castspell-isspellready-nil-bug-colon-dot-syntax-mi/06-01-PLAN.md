<objective>
Fix the colon/dot syntax mismatch in _castSpell that causes all Druid skill methods to silently fail. Replace 4 `self:xxx()` internal calls in entity/Player.lua with `obj.xxx()`, and replace 53 `self:_castSpell(...)` calls in classes/druid/Druid.lua with `obj._castSpell(...)`. Add ~15 Category F selftest tests verifying metatable chain integrity. Zero logic changes — purely mechanical `self:` to `obj.` replacements.

Purpose: Fix a parameter misalignment bug where Lua colon syntax implicitly inserts `self` (class prototype table) as first argument, shifting all intended parameters right by one position. This caused `isSpellReady(table)` to always return false.
Output: 3 files modified (entity/Player.lua 4 lines, classes/druid/Druid.lua 53 lines, core/selftest.lua ~15 new test registrations).
</objective>
